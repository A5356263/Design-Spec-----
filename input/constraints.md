# 约束条件
## 协议约束
- 必须输出为当前项目使用的扁平 page json
- 必须遵守 root / elements / type / props / children 结构
- 不允许私自扩展协议结构

## 组件约束
- 必须使用当前项目允许的 Ant Lite type
- 不允许使用 catalog 中未声明的 type
- 不允许使用 catalog 中未声明的 props

## 主链路约束
- 必须遵守三阶段推进
- 不允许跳过阶段直接生成最终页
- 不允许在后阶段推翻前阶段已通过结构

## 结构约束
- 页面固定为：顶部区 + 左右主体区 + 底部区
- 左侧为主要填写区
- 右侧为辅助信息区
- 不允许改成单栏
- 不允许改成多标签页结构

## 测试约束
- 当前测试目标是验证结构收敛，不是验证创意发散
- 优先稳定、规整、可评估
- 优先生成可通过当前项目 acceptance 的结果
